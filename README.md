# Conditional SQM Calculator

This calculator is used to accurately estimate a location's actual darkness. Many sources that are usually used to find dark skies give out raw SQM data, not accounting for the conditions. This can lead to huge differences between the estimated light pollution and what it actually is like over there. This is why I have coded this tool, that can calculate the darkness of an area using atmospheric and other conditions such as transparency, airglow, aerosols, the moon, planets, etc...

# How to use this?

To calculate the actual darkness of locations, this calculator comes in handy. Simply insert the light pollution map's estimate in "Light Pollution Map SQM", and then, enter the details for your location, such as altitude, transparency, airglow, etc... You can use websites such as https://www.astrospheric.com to predict transparency and other conditions. The calculation will automatically be done when you change parameters.
